# ローグライクRPGを作ろう1
- ./web/index.htmlにリンクを作成（フォーマットをそろえる）
- ./web/css/rougelike-codex.css にcssを置く
- ./web/rougelike-codex.html を作成
- ゲーム本体は ./web/js/rougelike-codex.js　に置く
- assetsは./web/assets/rougelike/ の下にある
- 既存の他のjsコードは参考しないこと

## 基本仕様
- 迷宮探索型ローグライク
- モンスターは階数を進むほど強くなる
- Expを溜めるとレベルアップ
- レベルアップするとHPとMPとステータスが上がる
- ステータス INT（魔法力） STR（力） DEX（移動速度） VIT（抵抗力）
- ダンジョンは3D表示で、右下にマップ 中央下はステータス表示
- ダンジョンにはお宝あるけど、たまに罠がある 罠にかかるとステータスが下がったり、HPが減ったり、異常状態になる
- モンスターを倒すと何ドロップするかも
- HPが0になるとゲームオーバー
- 所持品には制限がある(10種類まで)
- スクロールは使い切りの魔法 炎、水、風、土、治癒、状態回復、ステータスバフがある
- ポーションはHP回復、MP回復、ステータス回復がある
- 装備は武器と防具があるぞ 武器は攻撃力、防具は防御力を上げる
  - 武器は剣、槍、弓、魔法の杖がある 防具は鎧、盾、ヘルメットがある
  - 装備はEキーで持ち替えできる
  - 装備はモンスターがドロップすることもある
  - 装備は商店で買うこともできる
  - 装備は売ることもできる
  - 装備は捨てることもできる
  - 武器ごとにステータス加算が異なる
- 魔法
  - 魔法は攻撃系、状態系、位置制御系がある
  - 攻撃系はファイアボール、アイスシャード、ライトニングなどがある
  - 状態系はスリープ、パラライズ、ブラインドなどがある
  - 位置制御系はテレポート、フリーズ、グラビティなどがある
- 罠
  - テレポータ：どこかに飛ばされる
  - 毒針：ステータス毒になる
  - 落とし穴：ダメージを受けて一階下に落ちる
  - 爆発物：ダメージを受ける（DEXで回避可能）
  - アラーム：モンスターが出現する
  - 隠し針： 見えない罠。発動するとHPが減る(DEX/INTで回避可能)
- 罠はDEXが高くないとわからないかもしれない

- 宿屋　地上に戻ると宿屋がある。寝るとHPとMPとステータスが全開
- 商店　武器や魔法やスクロール、ポーションを買ったり、お宝を売ったりする

- 階段
  - 上り階段 上に昇る BF1から上に戻ると宿屋と商店がある
  - 下り階段 下に降りる 最下層にはボスがいるぞ

## レベルアップ
- プレイヤーは経験値を溜めるとレベルアップするぞ
- レベルアップするとHPとMPとステータスが上がるぞ
- requiredExp = 100 * level^2
- HP = 100 + level * (random(1, 20)) * class + VIT * 5
- MP = 50 + level * (random(1, 10)) * class + INT * 5
- STR = 5 + level * (random(1, 3)) * class + INT * 0.5
- DEX = 5 + level * (random(1, 3)) * class + INT * 0.5
- VIT = 5 + level * (random(1, 3)) * class + INT * 0.5

## クラス
- 戦士：HPとSTRが高い、MPが低い、VITが高い
  - HP = 1.2倍、STR = 1.5倍、VIT = 1.2倍 MP = 0.5倍
- 魔法使い：HPとMPが高い、STRが低い、INTが高い
  - HP = 0.8倍、MP = 1.5倍、STR = 0.5倍、INT = 1.5倍
- 盗賊：HPとDEXが高い、MPが低い、VITが低い
  - HP = 0.9倍、DEX = 1.5倍、VIT = 0.8倍、MP = 0.5倍
- 僧侶：HPとMPが高い、STRが低い、INTが高い、VITが高い
  - HP = 1.0倍、MP = 1.5倍、STR = 0.5倍、INT = 1.5倍、VIT = 1.2倍
- 冒険者：バランス型
  - HP = 1.0倍、MP = 1.0倍、STR = 1.0倍、DEX = 1.0倍、VIT = 1.0倍

## ダンジョンの生成
- ダンジョンは階層構造で、各階層は複数の部屋と通路で構成される
- 部屋はランダムな大きさと位置で生成される
- 部屋同士は通路で繋がれる
- 部屋と通路にはランダムにモンスターやアイテム、罠が配置される
- ダンジョンの最下層には『勝利のメダル』があるぞ(深さは10-20階層)
- ダンジョンの生成アルゴリズムはBSPツリー法や部屋通路法などがあるぞ
- ダンジョンの生成はゲーム開始時に行われるぞ
- 階段は部屋の中にランダムに配置される。階段を踏むと次の階層に進む。ダンジョンの各場所から階段へは必ず行けるようにする
- 隠し扉は部屋の壁にランダムに配置される。隠し扉を見つけると隠し部屋に入れる、隠し部屋にはレアアイテムや強力なモンスターがいる

## モンスターの強さ
- モンスターはダンジョンを徘徊している
- モンスターの強さは階層に応じて増加する
- モンスターの強さはHP、攻撃力、防御力、経験値で表される
- モンスターの強さはランダムに変動する
- モンスターの強さは種類によって異なる
- モンスターの種類はスライム、ゴブリン、オーク、ドラゴンなどがいる
- モンスターの種類は階層によって異なるぞ、下の階層ほど強いモンスターが出現する
- モンスターの種類はランダムに変動する
- モンスターはランダムにポップする

## 戦闘の計算
- 戦闘はターン制で、プレイヤーとモンスターが交互に攻撃する
- 攻撃は命中判定があり、命中するとダメージが入る
- ダメージは攻撃力と防御力の差分で計算される
- プレイヤーがモンスターを倒すと経験値とドロップアイテムが入る
- プレイヤーがHP0になるとゲームオーバー
- 細かい計算式
  - 命中率 = 0.75 + (攻撃側のDEX - 防御側のDEX) * 0.05
  - ダメージ = max(0, 攻撃力 - 防御力) * (1 + (攻撃側のSTR - 防御側のVIT) * 0.1) * random(0.85, 1.15)
  - 経験値 = モンスターのレベル * 10

## 戦闘
- プレイヤーがモンスターと接触すると戦闘が始まる
- 戦闘はターン制で、プレイヤーとモンスターが交互に攻撃する
- 攻撃は命中判定があり、命中するとダメージが入る
- ダメージは攻撃力と防御力の差分で計算される
- プレイヤーがモンスターを倒すと経験値とドロップアイテムが入る
- プレイヤーがHP0になるとゲームオーバー
- コマンド
  - 攻撃：通常攻撃をする
  - 魔法：MPを消費して魔法を使う
  - アイテム：所持品からアイテムを使う
  - 逃げる：戦闘から逃げる（成功率はDEXに依存）
  - スクロール：所持品からスクロールを使う
  - 装備：装備を変更する
  - 防御：次のターンまで防御力が上がる（敵の攻撃を軽減する）
  - 特殊攻撃：特定の条件を満たすと使える強力な攻撃（例：HPが半分以下のとき、特定の装備をしているときなど）
